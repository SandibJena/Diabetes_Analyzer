How To Read The Options

Each dropdown shows both short code and full meaning (example: H - High).

H: High

L: Low

N: Normal

I: Impaired vision

P: Present / Positive

A: Absent / Negative

D: Diabetic range (OGTT)

Y: Young or Years

M: Middle-aged or Months

E: Elderly

B: Below normal weight

W: Weeks

This tool is for awareness only and is not a medical diagnosis.
