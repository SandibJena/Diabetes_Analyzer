# Security Policy

## Supported Versions

Security fixes are provided for the latest `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue first.

Report it privately with the title prefix `SECURITY:` in GitHub Discussions or by contacting the repository owner directly.

Please include:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

You can expect an acknowledgment within 72 hours and follow-up after triage.
